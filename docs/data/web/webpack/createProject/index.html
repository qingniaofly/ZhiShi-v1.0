创建项目

介绍webpack的创建项目（基础版）

1、新建demo2文件夹，控制台进入该目录

执行 npm init

生成package.json

2、添加webpack（备用）、express、html-webpack-plugin包

执行 npm install webpack express html-webpack-plugin -s

3、新建app和webpack（备用）文件夹，目录结构如下

    
        index.js：开发模式入口（npm start 入口）
        app.js：webpack打包入口（备用）
        index.html：项目启动页面
        webpack.dev.conf.js：开发模式webpack配置（备用）
        webpack.prod.conf.js：生产模式webpack配置（备用）

4、修改package.json

    
        "start": "node app/index.js",
        "dev": "webpack --config webpack.dev.conf.js --color -p --progress --hide-modules --display-optimization-bailout",
        "build": "webpack --config webpack.prod.conf.js --color -p --progress --hide-modules --display-optimization-bailout"

5、index.html内添加内容

    
        <!DOCTYPE html>
        <html lang="en">
            <head >
                <meta charset="UTF-8">
                <title>Title</title>
            </head>
            <body>
            <div id="app">
                <h1<Hello World!!!</h1>
                </div>
            </body>
        </html>

6、index.js内添加内容

    
const express = require('express')
const app = express()

const fs = require('fs')
const path = require('path')

app.get('*', (req, res) => {
        fs.readFile(path.resolve(process.cwd(), 'index.html'), (err, file) => {
            if (err) {
                res.sendStatus(404);
            } else {
                res.send(file.toString());
            }
   });
});

const host = "http://127.0.0.1"
const port = 8050
app.listen(port,async err=>{
    if(err){
        console.error(err)
    }else{
        console.log('server start ', host+ ":" + port)
    }
})

7、执行 npm start

浏览 http://127.0.0.1:8050
